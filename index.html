Nuvio Installer

Install Nuvio on iPhone or iPad using AltStore.

Step 1

Install AltStore first

You need AltStore on your iPhone or iPad before installing Nuvio. AltStore cannot be installed directly from Safari. You need a computer the first time.

Open AltStore Website
You must install AltServer on your Mac or PC, connect your iPhone/iPad once, and install AltStore before continuing.
Step 2

Download Nuvio IPA

After AltStore is installed on your device, download the Nuvio IPA file.

Download Nuvio IPA

The file will usually go to Files > Downloads.

Step 3

Install Nuvio from AltStore

  1. Open AltStore on your iPhone or iPad.
  2. Tap the + button.
  3. Select the downloaded Nuvio IPA file.
  4. Wait for AltStore to finish installing the app.
If Nuvio was already installed before, AltStore may ask whether you want to keep the old version or replace it. In most cases, choose Remove App so AltStore can manage it properly.

Important

  • AltStore needs a computer for the first installation.
  • With a free Apple ID, apps normally need to be refreshed every 7 days.
  • If AltStore is already set up correctly, refreshing can happen automatically.
This installer page provides the IPA file and setup instructions. It does not bypass Apple's installation requirements.
